# ABSOLUTO
Retorna o valor absoluto de um número. O argumento pode ser um inteiro, um número de ponto flutuante ou um
objeto implementando __abs__(). Se o argumento é um número complexo, sua magnitude é retornada.

```
absoluto(x)

```
